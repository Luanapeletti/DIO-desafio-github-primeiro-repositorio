O Git Bash é um terminal extendido para otimizar o uso do Git. 
O comando git status serve para ver o que está dentro da pasta. 
O comando git add . ou git add -a serve para adicionar algum arquivo novo na pasta, ele irá reconhecer no comando.
O comando git clone serve para clonar um repositório que se encontra no GitHub.
O comando git commit -m "" serve para incluir as novas alterações no GitHub e as aspas serve para colocarmos uma especificação do que estamos adicionando.
